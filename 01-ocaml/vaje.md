## Naloga 2
V OCamlu napišite funkcijo `while_loop`, ki sprejme `cond`, `f` in `x` ter se obnaša kot zanka:
```
while cond(x):
  x = f(x)
return x
```
Kaj je tip funkcije `while_loop`?

Kako bi problem rešili za zanko:
```
result = 0
for x in list:
  result += f(x)
return result
```
